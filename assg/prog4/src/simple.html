Notes on HTML 1.0

HTML tags indicate the font style, such as:

    bold italic underline

The above is also an example of an unordered list. For coloration, use the font tag with the color option. To force line breaks, use the br tag.

Use the center tag to center text.


Tables are done using the table, tr, and td tags, as in the following 3x3 multiplication table:
1 2 3
2 4 6
3 6 9

You should remove any tags that appear in the HTML code, only preserving the text. Also, whitespace appearing in the HTML should not appear in the rendering; lines are only broken between paragraphs indicated by the p tag, and by manual line breaks indicated by the br tag.
